Portfolio API Documentation

Comprehensive API reference for the portfolio website

Authentication

Most API endpoints require authentication using an API key.

Header Authentication

X-API-Key: your-api-key-here

Query Parameter Authentication

GET /api/endpoint?api_key=your-api-key-here

Search API

Advanced Search

GET /api/search

Parameters

Parameter Type Required Description
q string Yes Search query (min 2 characters)
types array No Content types: blog, project, snippet
limit integer No Max results (default: 20, max: 50)

Example Request

curl -X GET "https://yoursite.com/api/search?q=python&types=blog,snippet&limit=10"

Example Response

{
  "results": [
    {
      "type": "blog",
      "id": 1,
      "title": "Advanced Python Techniques",
      "content": "Learn advanced Python programming...",
      "url": "/blog/advanced-python-techniques",
      "score": 15,
      "date": "2024-01-15T10:30:00Z"
    }
  ],
  "total": 1,
  "query": "python"
}

Request Body

}

Example Response

}

File Upload API

Upload File

POST /api/upload Admin Only

Form Data

Field Type Required Description
file file Yes File to upload (max 16MB)
public boolean No Make file publicly accessible

Supported File Types

Images: PNG, JPG, JPEG, GIF

Documents: PDF, TXT, DOC, DOCX

Archives: ZIP

Example Response

{
  "message": "File uploaded successfully",
  "file_id": 123,
  "filename": "abc123_document.pdf",
  "size": 1048576,
  "url": "/api/files/123"
}

Error Codes

Code Status Description
400 Bad Request Invalid request parameters
401 Unauthorized Missing or invalid API key
403 Forbidden Insufficient permissions
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred
